01 Project Overview

The Hand Gesture Controlled Robot is a six-axis pick-and-place robotic arm controlled wirelessly in real time via a Haptic Glove worn by the operator. Flex sensors on the glove capture finger bend angles, which are transmitted over Bluetooth and mapped directly to the servo joints of the 3D-printed robot arm.

Developed as the major project for the Diploma in Electronics at Temasek Polytechnic, it received a Certificate of Commendation, placing it in the top 10% of major projects for that cohort.

02 Objectives
  • 01Capture finger gestures in real time using a wearable flex-sensor glove
  • 02Transmit gesture data wirelessly with low latency to a remote robotic arm
  • 03Map gesture data to six servo joints for intuitive 1:1 control of the arm
  • 04Validate the system through real pick-and-place trials, aiding tasks for users with limited mobility

05 System Context

Master (Glove)

Haptic Glove with flex sensors on each finger, read by an Arduino Nano. Data transmitted wirelessly via HC05 Bluetooth module to the slave unit.

Slave (Robot Arm)

6-axis 3D-printed robotic arm with servo motors at each joint. Arduino Nano receives Bluetooth data and drives servos via PWM signals.

Communication

Master-Slave Bluetooth configuration using paired HC05 modules operating in AT command mode for low-latency wireless control.

Mechanical

Entire robot arm structure 3D printed in PLA, designed for lightweight, modular assembly with servo mounts at each of the six axes.

06 Technical Stack
  • Microcontrollers: Arduino Nano × 2 (master + slave)
  • Wireless: HC05 Bluetooth module pair (Master-Slave configuration)
  • Sensing: Flex sensors (one per finger) for gesture capture
  • Actuation: SG90/MG996R servo motors at six axes
  • Fabrication: FDM 3D printing (PLA) for the full arm structure
  • Language: C (Arduino firmware for master and slave)
07 Role & Contribution

Within the project team, I designed and fabricated the 3D-printed six-axis arm structure, implemented the flex-sensor ADC reading and servo PWM mapping on the master Arduino, and configured the HC05 Bluetooth modules for Master-Slave pairing. I also developed the C firmware for both master and slave units and led calibration of the gesture-to-joint mapping for intuitive control.
