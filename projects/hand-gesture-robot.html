Overview

The Hand Gesture Controlled Robot is a six-axis pick-and-place robotic arm controlled wirelessly in real time via a Haptic Glove worn by the operator. Flex sensors on the glove capture finger bend angles, which are transmitted over Bluetooth and mapped directly to the servo joints of the 3D-printed robot arm.

The project was developed as the major project for the Diploma in Electronics at Temasek Polytechnic and received a Certificate of Commendation — placing it in the top 10% of major projects for that cohort.

System Design

Master (Glove)

Haptic Glove with flex sensors on each finger, read by an Arduino Nano. Data transmitted wirelessly via HC05 Bluetooth module to the slave unit.

Slave (Robot Arm)

6-axis 3D-printed robotic arm with servo motors at each joint. Arduino Nano receives Bluetooth data and drives servos via PWM signals.

Communication

Master-Slave Bluetooth configuration using paired HC05 modules operating in AT command mode — achieves low-latency wireless control.

Mechanical

Entire robot arm structure 3D printed in PLA. Designed for lightweight, modular assembly with servo motor mounts at each of the six axes.

Key Technical Work
  • Designed and fabricated the 3D-printed 6-axis arm structure optimised for servo motor integration
  • Implemented flex sensor ADC reading and mapping to servo PWM range on the Arduino master
  • Configured HC05 Bluetooth modules in Master-Slave AT command pairing mode
  • Calibrated servo angle mapping to produce intuitive 1:1 gesture-to-joint control
  • Developed firmware in C for both master (glove) and slave (arm) Arduino Nano units
  • Tested pick-and-place operations with real objects, validating gesture control accuracy
Technology Stack
  • Microcontrollers: Arduino Nano × 2 (master + slave)
  • Wireless: HC05 Bluetooth module pair (Master-Slave configuration)
  • Sensing: Flex sensors (one per finger) for gesture capture
  • Actuation: SG90/MG996R servo motors at six axes
  • Fabrication: FDM 3D printing (PLA) for full arm structure
  • Language: C (Arduino firmware for master and slave)
