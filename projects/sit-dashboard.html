SIT Student Dashboard · Web App
Overview

The SIT Student Dashboard is a single-file, browser-based web application built as a practical daily tool for students at the Singapore Institute of Technology. It consolidates timetable views, campus alerts, and quick-access links into a unified interface — without requiring any backend or login infrastructure.

Built entirely with vanilla HTML, CSS, and JavaScript, the dashboard is designed for zero-dependency deployment: it runs as a static file, can be hosted on GitHub Pages, and persists data per student using namespaced localStorage keys.

View on GitHub
Features

Timetable Integration

Weekly timetable view with xSITE integration support — students can view their class schedule directly within the dashboard.

Drag & Drop Panels

Customisable dashboard layout with drag-and-drop panel repositioning, letting each student arrange their workspace to preference.

Dark Mode

Full dark/light mode toggle with persistent theme preference saved to localStorage across sessions.

Per-Student Namespacing

localStorage data is namespaced by student ID, enabling multiple users to share the same browser without data conflicts.

Campus Alerts

Alert panel for displaying campus announcements, deadlines, or important notices — editable directly in the interface.

Mobile Navigation

Responsive mobile navigation with a collapsible menu, optimised for use on phones and tablets on campus.

Technical Decisions
  • Single HTML file architecture — no build tools, no dependencies, no backend required
  • Vanilla JS drag-and-drop implementation using the HTML5 Drag and Drop API
  • localStorage used for all persistent state: theme, timetable data, panel layout, student ID namespace
  • CSS custom properties (variables) for theming — dark/light mode swap handled entirely in CSS
  • Designed for GitHub Pages hosting with no server-side logic needed
Technology Stack
  • Frontend: HTML5 · CSS3 (custom properties, grid, flexbox) · Vanilla JavaScript (ES6+)
  • Storage: Web Storage API (localStorage) with per-student namespacing
  • Hosting: GitHub Pages (static hosting, zero-config)
  • UI: Bootstrap Icons · Google Fonts · No external JS frameworks
Motivation

Built out of a genuine need — existing SIT student tools were scattered across different portals. The dashboard brings timetables, alerts, and links together in one place, and served as a personal project to develop web development skills alongside the Robotics Systems degree.
