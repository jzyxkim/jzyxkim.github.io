01 — Project Overview

The Intelligent Pallet & Optimization System is an autonomous warehouse measurement platform built to reduce manual labour and human error in cargo dimensioning and weighing. It pairs load-cell sensing with stereo camera vision to capture both the weight and 3D bounding dimensions of palletised cargo at a fixed checkpoint — with minimal operator intervention.

Developed as part of the SIT Robotics Systems capstone curriculum, the project targets smart manufacturing and logistics automation use cases where accurate, real-time cargo data is critical for warehouse management systems.

02 — Objectives
  • 01Capture accurate weight readings of palletised cargo using a load-cell array at a fixed checkpoint
  • 02Estimate 3D bounding dimensions (L × W × H) of cargo using stereo depth from a ZED2i camera
  • 03Synchronise and aggregate weight and vision data streams into a single timestamped payload
  • 04Validate the feasibility of low-cost stereo-camera dimensioning against manual measurement

05 — System Architecture

Weight Subsystem

Load cell array connected to an HX711 amplifier, read by a microcontroller and published to a ROS 2 topic for real-time weight capture.

Vision Subsystem

ZED2i stereo camera capturing RGB-D frames. Stereo depth estimation and point cloud processing compute the 3D bounding box of the cargo.

ROS 2 Integration

Node architecture with topic/service communication between sensor drivers, the vision pipeline, and the data aggregator node.

Output

Aggregated payload (weight, L × W × H dimensions, timestamp) formatted for export to a warehouse management dashboard.

06 — Technology Stack
  • Middleware: ROS 2 (Robot Operating System 2)
  • Vision: ZED2i Stereo Camera · ZED SDK · Point cloud processing
  • Sensing: Load cell array · HX711 24-bit ADC amplifier
  • Languages: Python (ROS 2 nodes, vision pipeline) · C (microcontroller firmware)
  • Compute: Edge compute node running Ubuntu with ROS 2 Humble
07 — Role & Contribution

I independently designed and built the full measurement pipeline for this capstone project: integrating the load cell and HX711 amplifier with a ROS 2 publisher node, developing the ZED2i stereo vision pipeline for depth estimation and bounding-box extraction, and writing the aggregator node that timestamps and merges both data streams into a single output payload for the warehouse dashboard.

08 — Verification & Validation

Depth Accuracy

Tested stereo depth estimation across varying warehouse lighting conditions to confirm consistent bounding-box accuracy.

Data Sync

Verified consistent timestamp alignment between the weight topic and vision topic across repeated runs.

Noise Filtering

Filtered point cloud noise to produce reliable bounding boxes for irregular cargo shapes.

Checkpoint Calibration

Calibrated the ZED2i camera for the fixed checkpoint geometry and confirmed measurement repeatability.

The system demonstrated autonomous measurement of pallet cargo weight and 3D dimensions with minimal human intervention, validating the feasibility of low-cost stereo-camera dimensioning for warehouse logistics checkpoints.
