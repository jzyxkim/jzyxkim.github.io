Intelligent Pallet & Optimization System

ROS 2Computer VisionPython ZED2i Stereo CameraLoad CellHX711
2024 SIT Capstone Jin Ziyu
Overview

The Intelligent Pallet & Optimization System is an autonomous warehouse measurement platform designed to reduce manual labour and human error in cargo dimensioning and weighing workflows. By integrating load cell sensing with stereo camera-based computer vision, the system captures both the weight and 3D bounding dimensions of palletised cargo at a fixed checkpoint — with minimal operator intervention.

The project was developed as part of the SIT Robotics Systems curriculum and targets smart manufacturing and logistics automation use cases, where accurate, real-time cargo data is critical for warehouse management systems.

System Architecture

Weight Subsystem

Load cell array connected to an HX711 amplifier, read by a microcontroller and published to a ROS 2 topic for real-time weight capture.

Vision Subsystem

ZED2i stereo camera capturing RGB-D frames. Stereo depth estimation and point cloud processing compute 3D bounding box dimensions of the pallet cargo.

ROS 2 Integration

Node architecture with topic/service communication between sensor drivers, the vision pipeline, and the data aggregator node.

Output

Aggregated payload (weight, L × W × H dimensions, timestamp) formatted for export to a warehouse management dashboard.

Key Technical Challenges
  • Achieving accurate stereo depth estimation under variable warehouse lighting conditions
  • Synchronising weight and vision data streams with consistent timestamping across ROS 2 nodes
  • Filtering point cloud noise to produce reliable bounding box estimates for irregular cargo shapes
  • Calibrating the ZED2i camera for the fixed checkpoint geometry to ensure measurement accuracy
  • Designing a robust load cell mounting that accounts for pallet positioning variance
Technology Stack
  • Middleware: ROS 2 (Robot Operating System 2)
  • Vision: ZED2i Stereo Camera · ZED SDK · Point cloud processing
  • Sensing: Load cell array · HX711 24-bit ADC amplifier
  • Languages: Python (ROS 2 nodes, vision pipeline) · C (microcontroller firmware)
  • Compute: Edge compute node running Ubuntu with ROS 2 Humble
Outcomes

The system demonstrated autonomous measurement of pallet cargo weight and 3D dimensions with minimal human intervention, validating the feasibility of low-cost stereo camera-based dimensioning for warehouse logistics checkpoints.
